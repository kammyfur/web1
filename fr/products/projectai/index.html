Use official versions (French)

Pour utiliser les versions officielles de Project AI, vous devez disposer d'un appareil compatible avec Google Assistant, tel qu'un appareil Android compatible, un Google Home, un Google Home Mini, ou une Smart TV compatible.

Vous devrez demander une question parmi les suivantes :
